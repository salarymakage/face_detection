### 🛠️ How to Set Up the Project

> **Note:** Only run the following steps. You can ignore all other files — they are just for testing and you will not understand them.

---

#### 1. Create a virtual environment

```bash
python -m venv env
```

---

#### 2. Activate the virtual environment

* **On Windows:**

  ```bash
  .\env\Scripts\activate
  ```

* **On macOS/Linux:**

  ```bash
  source env/bin/activate
  ```

---

#### 3. Install the required packages

```bash
pip install -r requirements.txt
```

---

#### 4. Run the project

```bash
python main.py
```
